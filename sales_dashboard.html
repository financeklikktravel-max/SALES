Data mapping. Region = Destination · Product category = Package (currently one type: Visa Processing) · Sales rep = Agent · Segment / Channel = Account (VSP / B2B). The source sheet has no dedicated fields for online/retail/wholesale/marketplace channel or customer-segment tiering.
Scope. VSP-channel transactions and their tied client entries are excluded from this dashboard by request — only B2B-channel activity is shown.
Coverage. Of 836 rows in the sheet, 34 have a usable date + client name (31 shown here after excluding VSP), and 3 have complete transaction financials (1 shown here after excluding VSP). Sections built on financials (KPIs, revenue/profit charts, target) reflect that 1 transaction; sections built on the fuller log (acquisition trend, new vs. returning) use the 31 remaining entries.
