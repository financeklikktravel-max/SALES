Data mapping. Region = Destination · Product = Package (raw text) · Product category = Package, grouped by keyword (e.g. "ALL IN…" → All-Inclusive, "…TICKET…" → Ticket) · Sales rep = Agent · Sales channel = the sheet's ADS/lead-source field, normalized (Walk-in, Ads, No Ads, B2B, Referral, Unspecified).
Scope. VSP-channel transactions are excluded from this dashboard by standing request — 99 rows were removed from the 5,284 valid transactions in the DAILY TRANSACTIONS tab, leaving 5,185.
Coverage. 5,185 real transactions from Jan 1, 2021 to Jul 15, 2026, with complete financials (revenue, cost, commission, profit). Every KPI, chart, and table on this page is computed live from that data — nothing is padded or estimated.
